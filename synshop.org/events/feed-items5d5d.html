Tinkercad 101: It's Back!

By Automated Importer

Friday May 24, 2013 7:00 pm

Tinkercad is back and we're teaching a class on it!

For those of you who do not know, Tinkercad is this awesome web based 3D CAD program. It was slated to go offline later this year. However, Autodesk just bought it and it will remain online. Yeah!

It is by far the easiest to use 3D CAD program that we have encountered.  Additionally, you can export the file to a format that can be used on the Shop's 3D printers.
