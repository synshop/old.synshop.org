Shopbot 100

By Automated Importer

Thursday Nov 6, 2014 6:00 pm

This class will guide students through the setup and use of the Shopbot. We will start from where the CAM setup class finishes with the Shopbot cut file prepared and after I explain and demonstrate the process each student will individually go through setting up the machine, load the file, change tools and make their own Synshop sign.

This introductory class will cover the following:
