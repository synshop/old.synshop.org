Shopbot 100

By Automated Importer

Thursday Nov 20, 2014 6:00 pm

This class will guide students through the setup and use of the Shopbot. We will start from where the CAM setup class finishes with the Shopbot cut file prepared and after I explain and demonstrate the process each student will individually go through setting up the machine, load the file, change tools and make their own Synshop sign.

This introductory class will cover the following:

Shopbot operation: job setup, fixtures, tools, safety, cleanup

Job set up: coordinate reference, CNC principles, hot key shortcuts

Solidworks 101

By Automated Importer

Tuesday Oct 28, 2014 6:00 pm

Solidworks is the most popular design software for the manufacturing industry and comes with a wealth of tools and resources to suit all your needs. Creating 3d parts is a simple process that will be guided through in this introductory class.  You will learn how to draw your part in 2d, extrude it out into a 3d shape, cut features and assemble it before exporting in a compatible format for CNC, 3d printing or laser cutting.

14 Oct 2014

Drawing for CNC

By Automated Importer

Tuesday Oct 21, 2014 5:30 pm

In this class we will become familiar with the drawing tools in Partworks 3.5.

I will be covering how to use downloaded artwork to create designs that can be made with CNC and the laser as well as creating original designs within Partworks with an included set of drawing tools that are able to meet the needs of most 2d projects. You will learn about open and closed vectors, stray lines, duplicate lines, and how to make everything work together and look the way you want it to.
