21 Aug 2014

Bead Work Class: Loom Work

By Automated Importer

Sunday Sep 21, 2014 12:00 pm

Learn bead working from Elaine Longoria, a Native American from the Tubatulabal tribe.  In this looming class students will be involved in a hands on workshop to learn how to warp (set up a loom) and weave the beads to make a charming bracelet.  Elaine will also provide her tips and tricks for a beautiful finished look.  The loom stitch is used to decorate, make jewelery, headbands, and many other beading accents.  This is a great beginners class for those interested in learning how to bead.

http://www.meetup.com/synshop/events/202463132/

Shopbot CAM setup - Partworks 2D and 3D

By Automated Importer

Tuesday Sep 2, 2014 6:45 pm

In this class students will learn how to set up 2D and 3D job files: making them ready to take to the Shopbot.

Students will learn:
How to make files compatible
Importing, features, options and setup guidelines for material setup
Drawing within the program, tool path choices, and 3D setup all the way through to a completed file ready for the Shopbot.

Materials Required by Students:
Computer loaded with Software

Prerequisites:
None

Laser Cutting 101

By Automated Importer

Tuesday Aug 12, 2014 6:45 pm

Laser cutters are great for both prototyping and production. This comprehensive class will cover the following:


1) Basic maintenance, troubleshooting, setup and use 

2) Overview of 2D and 3D design software 

3) Design theory 

4) Basic joints and box design tools 

5) 'Hacks' and advanced design tips 

6) Application - Personalize and cut a Raspberry Pi snap-fit enclosure. Based on an Adafruit design.
