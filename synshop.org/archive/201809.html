Open Source and Security Night with the Greynoi.se Podcast

By Automated Importer

3D Printing 101

By Automated Importer

Thursday Sep 13, 2018 7:00 pm

This is a class which covers some of the 3D printing basics!

The class covers:

- Anatomy of a 3D printer
- Safety when operating the machine
- Storage and treatment of consumables
- Basic maintenance of the machine
- Troubleshooting a variety of potential failures
- Available software to send prints to the machine
- Tips for leveling the build plate
